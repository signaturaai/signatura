## Verification Checklist

When testing the scoring system:

### 1. Industry Fairness
- [ ] Nurse scores should NOT be lower than engineer just because it's not tech
- [ ] "Technical skills" should be recognized in ALL fields (medical procedures, teaching methods, financial modeling)
- [ ] Feedback uses industry-appropriate language

### 2. Score Accuracy
- [ ] Scores within ±1 point of expected values
- [ ] Evidence cites specific text from the input
- [ ] Suggestions are actionable and relevant

### 3. Industry Weights
- [ ] Healthcare: Social Skills, Integrity, Adaptability weighted higher
- [ ] Education: Communication, Creativity, Learning Agility weighted higher
- [ ] Retail: Social Skills, Motivation, Leadership weighted higher
- [ ] Technology: Job Knowledge, Problem-Solving, Learning Agility weighted higher
- [ ] Finance: Integrity, Problem-Solving, Job Knowledge weighted higher

### 4. Feedback Quality
- [ ] Uses appropriate terminology:
  - Healthcare: "patient care", "clinical", "triage"
  - Education: "curriculum", "student engagement", "classroom"
  - Retail: "customer service", "sales", "merchandising"
  - Technology: "system design", "code quality", "architecture"
  - Finance: "financial modeling", "compliance", "analysis"
